Text

AX 인터페이스의 절반은 텍스트입니다. AI 응답, 사용자 입력, 인용, 메타데이터 등 다양한 텍스트 유형이 같은 시스템 안에서 일관된 위계로 표현되어야 합니다. Pretendard Variable + Poppins 조합이 한·영 혼용 환경에서도 안정적인 가독성과 시각적 정렬을 제공합니다.

Anatomy

Usage

Mixed Script

사용자와 AI Agent 가 함께 작업하는 AX UI 는 Pretendard Variable 로 한·영 혼용 환경에서도 같은 베이스라인을 유지합니다.

한글과 영문이 섞인 문장에서도 동일한 위계와 베이스라인이 유지되도록 Pretendard Variable 을 본문 전역에 적용합니다. 디스플레이 영역만 Poppins 로 분리하여 영문 헤더의 무게감을 표현합니다.

AI Response Streaming

사용자 의도를 파악한 다음 적절한 도구를 호출하고 결과를 정리하여 응답합니다

AI 응답은 글자 단위로 점진적으로 채워지는 streaming 표현이 기본입니다. 사용자에게 진행감을 전달하고 응답이 멈춘 듯한 느낌을 방지합니다. Body M 사이즈와 1.5 line-height 가 긴 응답을 읽기에 가장 적합합니다.

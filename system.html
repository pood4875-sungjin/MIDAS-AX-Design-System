Design System

MIDAS AX Design System 을 소개하는 이 사이트 자체를 짓기 위해 사용한 디자인 시스템입니다.

사이트의 모든 페이지 (Get a start · Foundation · Components · Pattern) 가 여기 정의된 토큰, 레이아웃, 컴포넌트로 만들어졌습니다.

각 항목은 토큰 정의 / 시각 예시 / 마크업 템플릿 / 실제 사용 위치를 함께 표시하며, 라이트·다크 모드에서 동일한 마크업으로 작동합니다.

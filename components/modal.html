Modal

중요 정보 표시 또는 사용자 결정을 요구하는 오버레이 컨테이너입니다. 배경을 dim 처리하여 본문과 분리하고, ESC · 배경 클릭 · 닫기 버튼 세 가지 경로로 닫을 수 있습니다. 포커스를 모달 내부에 가두는 focus-trap 으로 키보드 접근성을 보장합니다.

변경사항을 저장할까요?

저장하지 않고 닫으면 작성한 내용이 사라집니다.

Anatomy

Backdrop + Container 구조. 헤더(제목+닫기), 본문, 푸터(액션) 세 영역으로 구성. 등장 시 fade-in + scale 0.96→1, 퇴장 역순.

Icon Button

라벨 없이 아이콘만으로 액션을 표현하는 컴팩트 버튼입니다. AI 응답 위 인라인 액션(복사·재생성·좋아요·싫어요), 툴바 액션, 모달 닫기 등에 사용합니다. 단독 사용 시 반드시 aria-label 로 의미를 전달합니다.

Anatomy

정사각 컨테이너 + 18px 아이콘 + 8/10/12 radius 가 기본 anatomy 입니다. Variants 는 Filled / Tinted / Ghost 셋으로, 컨텍스트에 맞춰 강조 정도를 조절합니다.

  1. 1. Container
  2. 2. Icon (18-20px)
  3. 3. Background
  4. 4. Border
  5. 5. Hover bg
  6. 6. Pressed bg
  7. 7. Disabled
  8. 8. Tooltip (선택)
  9. 9. Size SM / MD
  10. 10. aria-label (필수)

Usage

Variants

Filled 는 강한 강조, Tinted (fill-normal bg) 는 중간, Ghost 는 약한 강조. 같은 툴바 안에서는 한 가지 변형으로 통일합니다.

Touch Target

최소 터치 영역 44×44 px 를 보장합니다. 아이콘이 24px 더라도 컨테이너는 40px 이상으로 유지하여 모바일·태블릿 환경의 사용성을 확보합니다.

How to use

aria-label="Copy to clipboard"
Do

단독 아이콘 버튼에는 반드시 aria-label 또는 tooltip 으로 의미를 전달합니다.

aria-label 누락 — 의미 전달 X
Don't

라벨 없는 아이콘 버튼에 aria-label 을 빠뜨리지 않습니다. 스크린리더에서 '버튼' 이라고만 읽힙니다.

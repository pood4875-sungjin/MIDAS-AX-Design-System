Avatar

발화 주체를 시각적으로 표시하는 작은 원형 컴포넌트입니다. 사용자(이니셜·프로필 사진)와 AI(시스템 로고·gradient)를 즉시 구분하고, System(공지·자동화)는 별도 회색 톤으로 표현합니다. 채팅 인터페이스의 시각적 리듬을 만들고 위계를 명확히 합니다.

Anatomy

Circle container + Image / Initials / Icon 슬롯. Status dot 으로 온라인·자리비움 상태를 표현할 수 있으며, 그룹 사용 시 -10px overlap 으로 stack 표현.

How to use

첫 메시지
두 번째 메시지 — avatar 생략
Do

발화 직전 메시지 옆에만 avatar 를 표시합니다. 연속된 같은 발화자에게는 반복하지 않습니다.

메시지 1
메시지 2 — avatar 반복
Don't

모든 메시지마다 avatar 를 반복 표시하지 않습니다. 시각 노이즈와 공간 낭비입니다.

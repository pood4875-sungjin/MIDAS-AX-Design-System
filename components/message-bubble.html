Message Bubble

사용자와 AI 의 발화를 시각적으로 구분하는 채팅 메시지 컨테이너입니다. 발화 주체에 따라 정렬·배경·radius 가 달라지며 markdown 응답, 코드 블록, 인용, streaming 등 다양한 컨텐츠를 포함합니다. tail radius(4px)로 발화 주체의 위치를 시각적으로 가리킵니다.

Anatomy

Container + Sender alignment + Background + Tail radius. 사용자 메시지는 오른쪽 정렬·dark fill, AI 메시지는 왼쪽 정렬·light fill 로 즉시 구분됩니다.

AI 응답이 여기에 표시됩니다. 충분히 긴 텍스트도 자연스럽게 줄바꿈됩니다.

Usage

Sender Differentiation

사용자 메시지
AI 응답 메시지

User 는 오른쪽 / inverse-background / 우하단 tail radius 4. AI 는 왼쪽 / fill-normal / 좌하단 tail radius 4. 정렬·컬러·tail 셋이 함께 발화 주체를 명확히 합니다.

Long-form Content

긴 응답은 max-width 로 70 글자 안에 유지되어 가독성을 확보합니다. markdown · code block · 인용 모두 같은 bubble 안에 자연스럽게 표시됩니다.

AI 응답이 길어지면 max-width 를 70 글자 이내로 유지하여 가독성을 확보합니다. markdown rendering 시 heading · list · code block · quote 모두 같은 bubble 안에 자연스럽게 표시.

How to use

질문
응답
Do

발화 주체별 정렬·컬러·tail radius 의 일관성을 유지합니다.

중앙 정렬 — 발화 주체 불명확
tail radius 일관성 없음
Don't

사용자와 AI 메시지의 시각 위계를 비슷하게 만들지 않습니다. 발화 주체 구분이 흐려집니다.

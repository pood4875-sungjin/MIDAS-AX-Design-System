Text Composer

AI 프롬프트 입력의 핵심 컴포넌트입니다. 사용자가 자연어로 AI 와 대화하는 입구이며, 파일 첨부 · 음성 입력 · 전송 · 중지 등 AX 특유의 액션을 통합합니다. 멀티라인 입력과 Enter 전송 / Shift+Enter 줄바꿈 / Esc 취소 등 키보드 단축키를 지원합니다.

어떤 도움이 필요하신가요?

Anatomy

Input area + Action bar (좌측 tools / 우측 send) 구조. 생성 중에는 Send 가 Stop 으로 전환되어 사용자가 언제든 응답을 중단할 수 있습니다.

Usage

States

디자인 토큰 사용 방법을 설명해줘

Idle / Composing / Sending / Streaming / Error 다섯 상태. Streaming 중에는 Send 가 Stop 으로 바뀌고, 사용자가 다시 입력하면 새 turn 으로 처리됩니다.

Keyboard Shortcuts

Enter 전송 · Shift+Enter 줄바꿈

Enter 전송 · Shift+Enter 줄바꿈 · Esc 입력 취소 · / slash 메뉴 · @ mention. 자주 쓰는 단축키는 placeholder 또는 tooltip 으로 노출합니다.

How to use

생성 중...
Do

생성 중에는 Send 버튼을 Stop 으로 전환해 사용자가 언제든 중단할 수 있게 합니다.

생성 중 — 입력 비활성화
Don't

생성 중이라고 input 자체를 비활성화하지 않습니다. 다음 메시지를 미리 작성할 수 있게 둡니다.

Text Field

짧은 텍스트 입력 컨트롤입니다. 폼·설정 화면에서 사용하며 label · helper · error 4-state 를 지원합니다. 입력 의도와 형식을 placeholder 가 아닌 label 로 명시하여 입력 시점에도 컨텍스트가 유지되도록 합니다.

name@company.com

알림을 받을 이메일 주소를 입력해주세요

Anatomy

Label 상단, Helper / Error 하단 배치. focus 시 보더 컬러·두께 변경, error 시 status-negative 컬러로 명확히 표시합니다.

name@company.com

필수 입력 항목

  1. 1. Label
  2. 2. Input
  3. 3. Placeholder
  4. 4. Helper
  5. 5. Error
  6. 6. Required *
  7. 7. Leading icon
  8. 8. Trailing icon
  9. 9. Counter
  10. 10. Disabled

Usage

Input Types

user@example.com
type="email" — 모바일 키보드 @ 키 노출

email · tel · number 등 적절한 input type 으로 모바일 키보드 자동 전환과 자동완성·검증을 활성화합니다.

Validation Timing

invalid-email

올바른 이메일 형식이 아닙니다

에러는 blur 또는 submit 시점에 표시. 사용자가 타이핑 중인 동안에는 가급적 에러를 보이지 않습니다.

How to use

홍길동
Do

Label 로 입력 의도를 명시하고, placeholder 는 예시 보조용으로만 사용합니다.

User name

label 없음 — 입력 시 컨텍스트 사라짐

Don't

placeholder 만 두고 label 을 생략하지 않습니다. 입력 시 컨텍스트가 사라져 접근성이 깨집니다.

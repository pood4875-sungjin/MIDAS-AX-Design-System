Components

실제 사용자 화면에 렌더되는 재사용 가능한 UI 블록.

모두 토큰 사용 → 라이트·다크 자동 적응. 썸네일 안에서만 쓰이는 demo 헬퍼 (chat, code-block 등) 는 제외됨.

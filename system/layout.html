Layout

전역 레이아웃 컴포넌트 (GNB · SNB · TOC · Footer) + 페이지 그리드 3 변형.

모든 페이지가 동일한 레이아웃 구조를 공유하여 시각 일관성과 사용자 익숙함을 유지합니다.

GNB · Global Navigation

72px sticky 헤더. frosted glass bg (rgba 88% + blur 16). 모든 페이지 상단에 동일하게 적용.

위치: 모든 페이지의 <body> 최상단 · position: sticky; top: 0; z-index: 100;
<header class="gnb gnb--solid">
  <div class="gnb__inner">
    <a class="gnb__logo" href="../index.html">
      <img src="../assets/images/logo-sparkle.png" alt="" width="30" height="30" />
      <span>MIDAS AX System</span>
    </a>
    <nav class="gnb__nav" aria-label="Global">
      <a href="../get-started/index.html">Get a start</a>
      <a href="../foundation/index.html">Foundation</a>
      <a href="../components/index.html">Components</a>
      <a href="../pattern/index.html">Pattern</a>
      <button class="gnb__theme" type="button">...</button>
    </nav>
  </div>
</header>
클래스역할
.gnb외부 컨테이너 (sticky, 72px)
.gnb--solid내부 페이지용
.gnb__inner1400 max-width 정렬
.gnb__logo로고 (Poppins 20/600/-0.8 + 회전 sparkle)
.gnb__navnav links + theme 버튼
.gnb__theme다크모드 토글 (sun/moon 아이콘 자동 전환)

SNB · Sub Navigation

좌측 sticky 사이드바. 그리드 셀 210 + 내부 wrapper(.snb__inner) 가 좌우 +10 확장 + 우측 30 스크롤바 영역.

구조: .snb (210 시각폭) → .snb__inner (margin -10/-10, padding-right 30, overflow-y auto) → items
<aside class="snb" aria-label="...">
  <div class="snb__inner">
    <a class="snb__overview" href="index.html">
      <span>Overview</span>
      <span class="snb__overview-arrow" aria-hidden="true"><svg>...</svg></span>
    </a>

    <p class="snb__caption">Section Name</p>
    <ul class="snb__list">
      <li><a class="snb__item" href="..." [aria-current="page"]>
        Item
        <span class="snb__item-arrow"><svg>...</svg></span>
      </a></li>
    </ul>
  </div>
</aside>

4-State Interaction

StateBackgroundColorArrow
Defaulttransparentlabel-alternativehidden
Hoverfill-hover (4%)label-strongvisible
Active [aria-current="page"]fill-hover (4%)label-strongvisible
Active + Hoverfill-pressed (8%)label-strongvisible

TOC · Anchor

우측 sticky 앵커 (160px). scroll-spy 로 현재 viewport 섹션에 active 표시. 클릭 시 smooth scroll.

<aside class="toc" aria-label="목차">
  <p class="toc__title">목차</p>
  <ul class="toc__list">
    <li><a class="toc__item" href="#section-id">Section Title</a></li>
  </ul>
</aside>
JS: main.js 의 scroll-spy 가 자동으로 aria-current="true" 토글
